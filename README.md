
Android BluetoothLeGatt Sample
===================================

Sample Android-Bluetooth project modified to get BLE signal strength from three BLE transmitters and calculate the coordinates of the device which run this code. Trilateration algorithm used to calculate the coordinates of the receiver.

This is accurate for 80% and accuracy may change according to the receiver and the transmitters.
